# 稳妥少年·生信分析实用小工具平台

## 📊 项目简介

**稳妥少年·生信分析实用小工具平台**是一个基于R/Shiny构建的生物信息学交互式分析工具集，为生物医学研究人员提供高效、便捷的一站式分析解决方案。平台集成了分子动力学、转录组学、药物发现、蛋白质互作等多个生物信息学核心分析模块，无需编程基础即可通过网页界面完成专业级分析。

## 🚀 核心特性

- **🔧 多工具集成**: 整合10+个实用生信分析工具，覆盖主要分析场景
- **🌐 零代码操作**: 基于Shiny构建的交互式Web界面，无需编程经验
- **⚡ 实时分析**: 在线处理数据，即时可视化分析结果
- **📈 专业可视化**: 提供出版级图表输出（火山图、热图、KM曲线等）
- **💾 数据互通**: 支持多种数据格式输入，分析结果可下载复用

## 🛠️ 技术架构

```
前端: R/Shiny + HTML/CSS/JavaScript
后端: R语言 + 生物信息学专用包
可视化: ggplot2, plotly, pheatmap, RCircos等
核心算法: limma, DESeq2, ssGSEA, STRING API等
部署: Shiny Server / RStudio Connect
```

## 📋 工具列表与功能

### 1. **MD模拟数据分析**
- **功能**: 分子动力学XVG文件可视化
- **输入**: GROMACS输出文件
- **输出**: 交互式时间序列图

### 2. **疾病转录组数据集合平台**
- **功能**: GEO/TCGA数据一站式获取
- **数据源**: NCBI GEO, TCGA官方数据库
- **特色**: 预筛选高质量数据集

### 3. **中西药分子库平台**
- **子模块**:
  - 中药分子库：传统中药活性成分
  - 西药分子库：FDA批准药物数据库
- **应用**: 药物筛选、分子对接、重定位研究

### 4. **差异表达基因分析平台**
- **算法**: limma (微阵列) + DESeq2 (RNA-Seq)
- **可视化**: 火山图、热图、MA图
- **输出**: DEG列表 + 统计报告

### 5. **在线富集分析平台**
- **数据库**: GO (基因本体) + KEGG (通路)
- **图表**: 柱状图、气泡图、网络图
- **统计**: 超几何检验 + FDR校正

### 6. **蛋白质互作网络分析**
- **数据源**: STRING数据库API
- **功能**: PPI网络构建与可视化
- **输出**: 互作网络图 + 拓扑指标

### 7. **数据相关性分析**
- **方法**: Pearson/Spearman相关分析
- **应用**: 共表达网络构建
- **可视化**: 相关矩阵热图

### 8. **单基因生存分析**
- **数据**: TCGA临床生存数据
- **方法**: Kaplan-Meier生存分析
- **检验**: Log-rank检验
- **输出**: KM生存曲线 + HR值

### 9. **基因染色体定位分析**
- **工具**: RCircos染色体圈图
- **功能**: 基因位置可视化、同线性分析
- **输入**: 基因坐标信息

### 10. **免疫细胞浸润分析**
- **算法**: ssGSEA (单样本基因集富集分析)
- **细胞类型**: 28种免疫细胞亚型
- **应用**: 肿瘤微环境解析
